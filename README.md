# CODSOFT Internship Tasks

This repository contains Machine Learning projects completed as part of CODSOFT internship.

## 🔹 Task 1: Movie Genre Classification
- Predicts movie genre based on text data.
- Techniques used: Text processing, ML classification models.


## 🔹 Task 2: Credit Card Fraud Detection
- Detects fraudulent transactions using ML algorithms.
- Techniques used: Data preprocessing, Logistic Regression, Evaluation metrics.

## 🔹 Task 3: Spam SMS Detection
- Classifies SMS messages as Spam or Not Spam.
- Techniques used: NLP, TF-IDF, Naive Bayes.


## 🛠️ Technologies Used
- Python
- Machine Learning
- Pandas, NumPy, Scikit-learn
- Google Colab

## 📌 Conclusion
These projects demonstrate practical implementation of Machine Learning concepts in real-world scenarios.
